## yasashii版の導入

`yasashii-harness` は、Agentic Harnessの規律を保ったまま、報告と言葉遣いを非エンジニアにも追いやすくした版です。
Claude Codeでは次の3コマンドで導入・起動できます。

```text
/plugin marketplace add mtaiseeei/yasashii-harness
/plugin install harness@yasashii-harness
/harness 作りたいものを一言で説明
```

やさしさは、言葉遣い、3行報告、次の一手の提案にだけ加えます。Planner / Generator / Evaluator の分離、
評価閾値、根拠、記憶保護、回帰ゼロ許容は緩めません。
